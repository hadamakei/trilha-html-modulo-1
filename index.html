Este é um projeto do curso da DIO que consiste em apresentar um resumo básico de definições de tags HTML.

Tags <h1> <h6>

Definição: Os títulos HTML são definidos com as tags de <h1> até <h6> .A Tag <h1> define o título mais importante. A Tag <h6> define o título menos importante. Exemplos:

Isto é um título h1

Isto é um título h2

Isto é um título h3

Isto é um parágrafo

Texto Destacado

A <mark> tag define o texto que deve ser marcado ou destacado. Exemplo:

Texto em destaque.

Texto Pequeno

A <small> tag define textos menores (como direitos autorais e outros comentários paralelos). Exemplo:

Texto em tamanho menor.

Voltar

Tag HTML <i>

A <i> tag define uma parte do texto em um tom ou modo alternativo. O conteúdo interno normalmente é exibido em itálico. É frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc. Exemplo:

Texto em formato itálico.

Voltar

Tag HTML <u>

A <u> tag representa algum texto não articulado e com estilo diferente do texto normal, como palavras com erros ortográficos ou nomes próprios em texto chinês. O conteúdo interno normalmente é exibido com um sublinhado. Você pode alterar isso com CSS. Exemplo:

Palavra com erro de ortografiah.

Voltar

Tag HTML <strong>

A <strong> tag é usada para definir texto com grande importância. O conteúdo interno normalmente é exibido em negrito. Exemplo:

Palavra com importância.

Voltar

Tag HTML <ol>

A <ol> tag define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética. Exemplo:

  1. Coffee
  2. Tea
  3. Milk
Voltar

Tag HTML <ul>

A <ul> tag define uma lista não ordenada. Exemplo:

  • Café
  • Chá
  • Leite
Voltar

Tag HTML <li>

A <li> tag define um item das listas ordenadas, não ordenadas e menu. Exemplo:

  • Item da lista 1
  • Item da lista 2
  • Item da lista 3
Voltar

Tag HTML <a>

A <a> tag define um hiperlink, que é usado para vincular de uma página para outra. O atributo mais importante do < a> elemento é o href atributo , que indica o destino do link. Exemplo:

Link da W3Schools.com!

Voltar

Tag HTML <hr>

A <hr>tag define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico).O elemento é geralmente exibido como uma linha horizontal usada para separar conteúdo (ou definir uma alteração) em uma página HTML. Exemplo:

Conteúdo 1

Conteúdo 2

Voltar

Tag HTML <sub>

A <sub> tag define texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas. Exemplo:

Fórmula química da água H2O.

Voltar

Tag HTML <sup>

A <sup> tag define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé. Exemplo:

Referências bibliográficas nota de rodapé [1].

Voltar

Tag HTML <blockquote>

A <blockquote> tag especifica uma seção que é citada de outra fonte. Os navegadores geralmente recuam < blockquote>elementos. Exemplo:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.
Voltar

Tag HTML <del>

A <del> tag define o texto que foi excluído de um documento. Os navegadores geralmente riscam o texto excluído. Exemplo:

Texto que foi excluído.

Voltar

Tag HTML <abbr>

A <abbr> tag define uma abreviação ou um acrônimo, como "HTML", "CSS", "Sr.", "Dr.", "ASAP", "ATM". Exemplo:

A OMS foi fundada em 1948.
